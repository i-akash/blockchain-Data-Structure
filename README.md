# blockchain Data Structure
    
##### Instruction
        CMD  ----> Description
      
      1. sync ---> Synchronize Chain
      
      2. save ---> Save Chain
      
      3. show qt -----> Show Queue Transaction
      
      4. show st -----> Show Selected Transaction
      
      5. show ch -----> Show Chain
      
      6. select t ----->  Select Transaction
      
      7. add t ------> Add Transaction
      
      8. add b -------> Add Block
      
      9. exit ----> Exit
